### 🔴 Как выключить

1. **Останови JSON Server**  
   В терминале, где запущен `npx json-server --watch db.json --port 3000`, нажми **Ctrl + C**. Сервер остановится, порт 3000 освободится.

2. **Терминал с тестами можно просто закрыть**  
   После завершения `npm test` он сам завершился, дополнительно ничего глушить не надо.

Теперь всё выключено, можно спокойно закрывать окна терминала или редактор.

### 🟢 Как снова всё запустить

1. **Открой терминал** в папке проекта `mock-lab`.

   ```bash
   npm install
   ```

2. **Запусти JSON Server** (обязательно перед тестами API):
   ```bash
   npx json-server --watch db.json --port 3000
   ```
   Должен появиться довольный смайлик и список эндпоинтов. Не закрывай это окно.

3. **Во втором терминале** (или вкладке) в той же папке выполни:
   ```bash
   npm test
   ```
